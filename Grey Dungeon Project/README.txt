Grey Dungeon - это псевдо 3D шутер от первого лица, выполненный в монохромной стилистике
-----------------------------
РЕЖИМЫ
-----------------------------
Есть два режима:
1) Solo:
Для запуска этого режима просто запустите файл main
Условие победы - убиство враждебного солдата или нейтрального
Если ваше здоровье опускается ниже 0 - вы проиграли

2) Duo:
Для запуска этого режима сперва запустите файл server (причем только на одном устройстве, которое и будет хостом)
Для корректного подключения и запуска этого режима незабудьте прописать свой локальный IP (IPv4) в файле ipconfig.txt
Порт можно не менять тк он в большинстве случаев свободен
Если будет введен некорректный IP, то при запуске main автоматически запуститься Solo-режим
Условие победы - убиство враждебного игрока (npc просто будет мешать ближайшему игроку)
Если ваше здоровье опускается ниже 0 - вы проиграли
-----------------------------
СБОРКИ
-----------------------------
На данный момент есть две сборки:
1) Python-версия:
Собственно, обычные .py файлы, алгоритм запуска смотри выше.
Версия Python - 3.8.3, но и на 6 и на 7 все работало
Использованные сторонние библиотеки:
pygame - 2.0.0.dev2, это еще не release версия, но она работа быстрее, вот и использовал её (1.9.6 будет работать спокойно, проблем не наблюдалось)
numba - 0.51.2, это just in time компилятор python кода в машинный. Позволяет получить прирост производительности в несколько десятков раз

2) .EXE-версия:
Скомпилированная с помощью pyinstaller версия, где все библиотеки уже добавлены, так что нет нужды в их установке (как и самого python)
Работает точно также, как и Python-версия, однако иногда теряешься во всех файлах (хотя для исполняемых файлов сделаны отдельный иконки)
Так что советую создать ярлыки вне этой папки
-----------------------------
WARNING
Иногда антивирус может ругаться и помещать в карантие файл server, из-за его локального подключения, 
поэтому либо добавьте его в whitelist, либо отключите на время антивирус
Ну или играйте в Solo-режим, там таких проблем замечано не было.
-----------------------------
P.S. Duo-режим будет работать в любой локальной сети. Например, мы проверяли через hamachi и все работало вполне стабильно,
однако из-за большого расстояния у клиента будет сниженное количество кадров (в районе 20, вместо 60-90)

Также при первом запуске могут быть небольшие фризы и просед кадров, однако во время последующих запусков они пропадут (библиотека numba поместит в cache некоторые функции)
